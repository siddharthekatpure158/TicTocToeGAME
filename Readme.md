# Tic Tac Toe Game

A simple **Tic Tac Toe** game implemented in Java.  
Two players can play against each other on the same computer.

---

## 🕹 Features

- Classic 3x3 Tic Tac Toe grid
- Two-player mode (no AI for now)
- Input validation to prevent overwriting moves
- Declares the winner or tie automatically

---

## 💻 Technologies Used

- Java 8+
- Console-based application
